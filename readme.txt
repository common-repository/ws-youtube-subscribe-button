== Description ==

This plugin provides a small button displayed on your blog to help users easily subscribe to your YouTube channel. 

The YouTube Subscribe Button lets you add a one-click Subscribe button to any page. The button lets people subscribe to your YouTube channel without having to leave your site to either log in to YouTube or confirm their subscriptions.

<strong>Features</strong> 

* Easy to install and Configure
* User friendly and easy to use.
* Simply Add Subscribe Button
* Change language of subscribe button
* Easy to add shortcode anywhere in your wordpress site.

== Installation ==

1. Download [WS Youtube Subscribe Button](https://wordpress.org/plugins/ws-youtube-subscribe-button/) plugin.   
2. Upload the WS Youtube Subscribe Button plugin from your admin panel.    
3. Activate the WS Youtube Subscribe Button plugin. 
4. Add WS Youtube Subscribe Button widget to your sidebar.
